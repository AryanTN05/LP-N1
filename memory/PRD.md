# Aryan TN - AI Automation Landing Page

## Original Problem Statement
Build a freelancing landing page for Aryan TN who provides AI automation services for B2B lead generation. Reference site: apixsystems.com. Dark minimal theme with sections: Hero, Pain Points, Solution, Results, Why Me, Process, FAQ, Footer.

## Architecture
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI (minimal - no backend logic needed for landing page)
- **Database**: MongoDB (not used for landing page)
- **Hosting**: Self-hosted

## User Persona
- B2B company founders, sales leaders, decision-makers in US/Europe/UAE
- Looking for AI-powered lead generation solutions
- Frustrated with manual prospecting and agencies with no results

## Core Requirements (Static)
- [x] Hero section with status badge, heading, CTA, tool logos (Claude, n8n, Clay, Apollo.io, Slack)
- [x] Pain Points "Is This You?" section with 4 cards
- [x] Solution section with bento grid layout
- [x] Results section with checkmark list
- [x] Why Work With Me section with 5 feature cards
- [x] How It Works process timeline with 4 steps
- [x] FAQ section with 6 accordion items (Shadcn Accordion)
- [x] Footer with CTA band, LinkedIn, email links
- [x] Dark minimal theme (#09090b background)
- [x] Mobile responsive
- [x] All CTAs link to cal.com/aryantn01/30min
